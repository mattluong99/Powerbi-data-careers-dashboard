# Data Careers Survey Dashboard

## 📌 Project Overview

This project analyzes survey data from data professionals to uncover insights into:

* Salary trends across roles
* Most popular programming languages
* Difficulty entering the data field
* Work-life balance and salary satisfaction

## 🎯 Objective

To explore key factors influencing careers in data and present insights through an interactive Power BI dashboard.

## 🛠 Tools & Technologies

* Power BI (Data Visualization)
* Power Query (Data Cleaning)
* DAX (Data Modeling & Measures)

## 📷 Dashboard Preview

![Dashboard](images/dashboard-preview.png)

## 📊 Key Insights

* Data Scientists have the highest average salaries
* Python is the most widely used programming language
* A large portion of respondents find it difficult to enter data careers
* Work-life balance satisfaction is higher than salary satisfaction
